# rebootToUEFI

Simple batch script to reboot a Windows 10/11 machine into UEFI firmware settings.

## Note

This script will only work on Windows 10/11 machines that support UEFI firmware. It will not work on older machines that use BIOS firmware.

## Admninistrative Privileges

This script requires administrative privileges to run. Make sure to run the script as an administrator to ensure it works correctly.
